# SimpleStorage Smart Contract

Basic Solidity contract for learning:
- structs
- arrays
- mappings
- CRUD operations
- Remix VM
- EVM compatibility

Built while learning Web3 + Solidity.

## Tech
- Solidity ^0.8.31
- Remix IDE
- Ethereum VM (Shanghai)

## Features
- Store favorite number
- Add people
- Retrieve people
- Struct storage
